# Roadmaps Polish & UX Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add unlimited undo/redo (snapshot approach per mutation), a full keyboard shortcut set, and hover tooltips to all Toolbar buttons.

**Architecture:** History is managed in Canvas.jsx via a `historyRef` (useRef) holding `{ past: [], future: [] }` arrays of board snapshots (`{ nodes, edges, groups }`). A wrapper `applyUpdate(fn)` replaces direct `onUpdate` calls for all recordable mutations — it snapshots the current board before applying the change. Drag operations snapshot at drag-start and record a single entry at drag-end. Keyboard shortcuts extend the existing `useEffect` listener in Canvas.jsx. Tooltips are native `title` attributes on Toolbar buttons.

**Tech Stack:** React 18, existing codebase — no new dependencies

```js
// Record a mutation in history and apply it
const applyUpdate = fn => {
  const snapshot = { nodes: board.nodes, edges: board.edges, groups: board.groups };
  historyRef.current.past.push(snapshot);
  historyRef.current.future = [];
  setCanUndo(true);
  setCanRedo(false);
  onUpdate(fn);
};

const undo = () => {
  const { past, future } = historyRef.current;
  if (!past.length) return;
  const snapshot = past.pop();
  future.push({ nodes: board.nodes, edges: board.edges, groups: board.groups });
  setCanUndo(past.length > 0);
  setCanRedo(true);
  onUpdate(b => ({ ...b, ...snapshot }));
};

const redo = () => {
  const { past, future } = historyRef.current;
  if (!future.length) return;
  const snapshot = future.pop();
  past.push({ nodes: board.nodes, edges: board.edges, groups: board.groups });
  setCanUndo(true);
  setCanRedo(future.length > 0);
  onUpdate(b => ({ ...b, ...snapshot }));
};
```
